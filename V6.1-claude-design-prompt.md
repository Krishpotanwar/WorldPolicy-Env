## What V6.1 Adds — One-Line Summary

A **Debate Chamber** that sits beside the globe in Split Mode. Seven AI agents — USA, China, Russia, India, North Korea, Saudi Arabia, and UNESCO — speak in real time when a crisis fires. Each country is a full persona with memory, alliances, and grudges. A live Country P&L ledger and a separate Company P&L strip tick red/green as each agent speaks. UNESCO acts as a neutral mediator. The MAPPO policy still chooses actions, but the LLM-driven debate explains *why* — and a vote can confirm or modify the action.

## Component Specs

Produce each of the following as a single `.tsx` file under `frontend/src/components/`. Use TypeScript, functional components, hooks only. No state libraries — props + `useState` + `useEffect` + one `useEventSource` custom hook for SSE.

### 1. `AgentPortraitStrip.tsx`

A horizontal row of 7 circular portraits at the top of Split Mode.

**Props:**
```typescript
interface AgentPortrait {
  id: 'USA' | 'CHN' | 'RUS' | 'IND' | 'DPRK' | 'SAU' | 'UNESCO';
  name: string;
  tint: string;              // CSS color var or literal
  isActiveSpeaker: boolean;
  isInvolvedInCrisis: boolean;
  relationshipStatus?: 'ally' | 'rival' | 'neutral';  // relative to focused country
  lastUtteranceStep?: number;
}
interface AgentPortraitStripProps {
  portraits: AgentPortrait[];
  focusedCountryId: string | null;    // which portrait the user clicked
  onHover: (id: string | null) => void;
  onClick: (id: string) => void;
}
```

**Visual design:**
- Portrait = circular glass disc, 72px diameter, country flag/emblem rendered as a subtle CSS gradient or abbreviated 2-letter code in a monospace stencil font. No flag image files — purely CSS.
- Default state: soft glow ring in the country tint at 20% opacity.
- **Hover state**: scale to 1.08, ring opacity → 50%, tint bloom grows, subtle lift shadow. Transition 200ms cubic-bezier(0.16, 1, 0.3, 1).
- **Active speaker**: ring pulses at 1.5s cadence, opacity oscillates 40% → 90%, a small `● LIVE` LED sits at the top-right corner of the portrait (use `--led-green` with `--led-green-glow` and `led-pulse` animation).
- **Involved in crisis**: portrait gets a second outer ring in `--accent-amber` at 30% opacity. This is independent of the active-speaker ring — can stack.
- **Not involved**: dim to 55% opacity. Desaturated. Fades to the background.
- **Focused (clicked)**: scale 1.15, tint ring at 100%, and a short vertical tick mark under the portrait pointing down to the transcript panel (indicates "you are viewing this agent's opinions").

**Ordering:** USA, China, Russia, India, DPRK, Saudi, then a slight gap (16px), then UNESCO — so the mediator is visually separated. This reinforces the "non-voting" framing.

**UNESCO portrait specifics:** replace the stencil code with a small SVG laurel ring, teal-tinted. Under the name, in 9px uppercase mono: `MEDIATOR · NON-VOTING`.

### 2. `DebateTranscriptPanel.tsx`

The scrolling transcript on the right side of Split Mode.

**Props:**
```typescript
interface DebateUtterance {
  step: number;
  speakerId: string;
  speakerName: string;
  speakerTint: string;
  text: string;
  stance: 'support' | 'oppose' | 'modify' | 'neutral' | 'mediate';
  pnlDeltas: Record<string, number>;     // { USA: -0.02, CHN: +0.01, ... } fired when this utterance lands
  timestamp: number;
}
interface DebateTranscriptPanelProps {
  utterances: DebateUtterance[];
  focusedCountryId: string | null;       // when set, filters to only utterances from this country or about it
  activeSpeakerId: string | null;
  voteTally: { support: number; oppose: number; modify: number } | null;
}
```

**Visual design — scrolling glass document:**

Each utterance is a row composed of:
- **Left rail** (4px wide): vertical bar in `speakerTint` at 80% opacity, full row height. Creates a color-coded margin like a speaker-identified debate transcript.
- **Speaker header** (row 1): speaker name in `--font-mono` 11px uppercase, letter-spacing 1.2px, color = speakerTint. To the right: small stance pill — `SUPPORTS` (green), `OPPOSES` (red), `MODIFIES` (amber), `MEDIATES` (teal), `NEUTRAL` (slate). Pill is 3px radius, 9px font, 2px 6px padding, bg = stance color at 15% alpha, border = stance color at 40% alpha.
- **Utterance body** (row 2): text in `--font-ui` 13px, line-height 1.5, color rgba(255,255,255,0.85). Max 3 lines truncated with ellipsis unless this is the focused speaker.
- **Meta line** (row 3, 9px mono slate): `step 47 · 1.8s after MAPPO action · pnlΔ: USA −0.02, CHN +0.01, IND +0.00`
- **Divider below row**: 1px line, gradient fade, matches V5 `.emergent-panel .panel-divider`.

**Active speaker row**: gets a soft bg wash = speakerTint at 6% alpha, a stronger left rail (6px, 100% opacity), and a subtle right-edge glow. When a new utterance appears, animate in from `translateY(12px) opacity:0` to `translateY(0) opacity:1` over 400ms.

**Auto-scroll** to newest utterance unless user has scrolled up manually (detect via scrollTop offset from bottom > 200px). When user has scrolled up, show a small floating "↓ New speeches" pill at the bottom of the panel.

**Focused country filter:** when a portrait is clicked, the transcript filters to:
1. All utterances *by* the focused country
2. Plus all utterances *by other countries* where the focused country was mentioned by name in `text` (case-insensitive substring match)
3. Plus all utterances marked with `stance: 'mediate'` (UNESCO is always shown because mediation applies broadly)

A small header above the transcript reads: `Viewing: USA — 4 of 12 speeches · Clear filter`. The "Clear filter" is a text button that resets focus.

**Vote tally** at the bottom: when `voteTally !== null`, show three horizontal bars (Support / Oppose / Modify) with counts. Animate bar width from 0 to final over 1200ms when tally arrives. If vote passes (support > oppose), show a small green checkmark LED; if it fails, red X LED.

### 3. `CountryPnLLedger.tsx`

Live-ticking ledger of 7 country rows × 7 metric columns.

**Props:**
```typescript
interface CountryPnL {
  countryId: string;
  countryName: string;
  tint: string;
  metrics: {
    gdp: number;         // delta from baseline, signed
    jobs: number;
    energy: number;
    influence: number;
    welfare: number;
    heritage: number;    // UNESCO-relevant
    military: number;
  };
  deltasSinceLastTick: Record<keyof CountryPnL['metrics'], number>;  // for flash animation
}
interface CountryPnLLedgerProps {
  rows: CountryPnL[];
  activeSpeakerId: string | null;   // row of active speaker gets highlighted
}
```

**Visual design — skeuomorphic trading terminal:**

Render as a dense table, `--font-mono` 10px, row height 24px.

- Header row: 9px uppercase mono, letter-spacing 1px, color rgba(255,255,255,0.35), border-bottom 1px rgba(255,255,255,0.08).
- Data rows: country code on left (USA, CHN, RUS, IND, DPRK, SAU) in speaker tint, then 7 metric columns right-aligned numeric.
- Numeric cells: show values signed, e.g. `+0.042`, `-0.017`. Positive → green tint background at 3% alpha. Negative → red tint background at 3% alpha. Zero → no wash.
- **Flash animation**: when `deltasSinceLastTick[metric] !== 0`, flash that cell's background at speakerTint at 40% alpha for 600ms, then decay back to the sign wash. This creates the "red/green tick" effect as each agent speaks.
- **Active speaker row highlight**: left border 3px in speakerTint, full row bg at speakerTint 5% alpha.
- UNESCO row: show only `heritage` column filled; other columns show `—` in rgba(255,255,255,0.2). UNESCO has no national P&L — make this visually clear.

A small header above the ledger: `COUNTRY P&L · real-time delta per debate round` — 9px mono uppercase, same style as the V5 `.emergent-panel .panel-header`.

## Interaction Rules

### Hover / Click behavior on portraits (from Q3)

1. **Hover on portrait**: portrait scales 1.08, tint ring brightens. A small tooltip appears 8px below the portrait showing: agent name, current stance on active crisis ("Supports aid dispatch"), last utterance step. 180ms appearance delay.

2. **Click on portrait**: portrait becomes "focused." Transcript panel filters per the rules in §2 above. Globe highlights the clicked country plus any country mentioned in its utterances (amber ring on globe node). Other non-involved countries on globe dim to 55% opacity.

3. **Click the focused portrait again, or click elsewhere**: focus clears. Transcript returns to full stream. Globe dim-outs clear.

4. **Click a country directly on the globe**: same as clicking its portrait.

### Involvement highlighting (from Q3)

Agents "involved" in the active crisis are determined by the backend, surfaced via `GET /crisis-involvement/{crisis_id}` → `{ involved: ['USA', 'CHN'], peripheral: ['IND', 'UNESCO'], uninvolved: ['RUS', 'DPRK', 'SAU'] }`.

- **Involved**: full saturation, amber outer ring.
- **Peripheral**: 80% opacity, no extra ring.
- **Uninvolved**: 55% opacity, desaturated.

UNESCO is always at least peripheral because mediation applies to all crises.

## Data flow

Add these custom hooks under `frontend/src/hooks/`:

- `useDebateStream()` → subscribes to `GET /stream/debate` SSE, returns an append-only array of `DebateUtterance` objects plus the current `activeSpeakerId`.
- `useCountryPnL()` → subscribes to `GET /stream/country-pnl`, returns the latest `CountryPnL[]` snapshot plus per-metric deltas since last tick.
- `useCompanyTicks()` → subscribes to `GET /stream/company-pnl`, returns `CompanyTick[]`.
- `useCrisisInvolvement(crisisId: string | null)` → fetches `GET /crisis-involvement/{id}`, returns the three buckets.
- `useLayoutMode()` → reads/writes `'globe' | 'split' | 'chamber'` to localStorage under key `wp:layoutMode`, default `'split'`.

Keep all these hooks thin — they should be essentially wrappers over a shared `useEventSource<T>(url: string)` primitive plus a `useState` latch.
